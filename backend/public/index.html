API Endpoints

GET /countries

Retrieve all countries

GET http://localhost:1008/countries

GET /roles

Retrieve all roles

GET http://localhost:1008/roles

GET /users

Retrieve users with filtering, sorting, and pagination

Query Parameters:

Page number

_page

Items per page

_limit

Total items (returned in response header)

X-Total-Count

Sort by field (e.g., lastName)

_sort

Sort order (asc/desc)

_order

Full-text search

q

Filter by country ID

country.id

Filter by role name

role.name

Example:

GET http://localhost:1008/users?_page=1&_limit=20&_sort=lastName&_order=asc

DELETE /users/:id

Delete a specific user

DELETE http://localhost:1008/users/1

To access and modify resources, you can use any HTTP method:

GET POST PUT PATCH DELETE OPTIONS
